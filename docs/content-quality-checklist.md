# 콘텐츠 품질 운영 체크리스트

이 문서는 운영자용입니다. 사이트 방문자에게 직접 노출하지 않습니다.

## 발행 전 기준

- 검색 수요가 있는 주제인지 확인한다.
- 제목에는 핵심 검색어를 자연스럽게 포함한다.
- 공식 확인 경로 또는 참고 기준을 함께 적는다.
- 단순 요약이 아니라 실제 지역 기준, 체크리스트, 비교표 중 하나 이상을 추가한다.
- 과장 표현, 클릭 유도 문구, 근거 없는 단정 표현을 제거한다.
- 같은 구조의 글을 반복 발행하지 않는다.

## 글 구조

1. 누가 읽으면 좋은 글인지 먼저 밝힌다.
2. 가장 중요한 결론을 앞부분에 배치한다.
3. 조건, 비용, 신청 방법, 준비물, 주의사항을 구분한다.
4. 표나 체크리스트를 포함한다.
5. 관련 글로 자연스럽게 내부 링크를 연결한다.
